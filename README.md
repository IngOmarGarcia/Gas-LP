# 🛢️ Sistema de Gestión para Distribuidora de Gas LP

## 📋 Descripción
Aplicación móvil y web desarrollada para facilitar la administración integral de una distribuidora de Gas LP. Permite llevar el control de todo el negocio de forma rápida, organizada y accesible desde cualquier dispositivo.

## ⚙️ Funcionalidades principales
- Registro, consulta y gestión de clientes
- Control de clientes y estados de servicios
- Registro de ventas, entregas y movimientos diarios
- Inicio de sesión seguro mediante cuentas de Google
- Envío automático de correos electrónicos (notificaciones, alertas y confirmaciones)

## 🛠️ Tecnologías y Herramientas utilizadas
- Framework de desarrollo: Ionic (Permite el funcionamiento en Android, iOS y Web con un solo código base)
- Gestor de contenido y lógica de negocio: Strapi (CMS de código abierto para creación de APIs y administración de datos)
- Base de Datos: PouchDB (Almacenamiento de información y sincronización de datos)
- Lenguajes: HTML, CSS, JavaScript / TypeScript

## ▶️ Modo de uso
1. Acceso mediante cuenta de Google autorizada
2. Navegación intuitiva entre los módulos de Pedidos, Clientes e Historial
3. Registro rápido de operaciones
4. Consulta de información en tiempo real
